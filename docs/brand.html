Brand Guide

Colors, typography, logo, wordmark, and image assets — shared by the web app and the docs site.

Using in Canva / design tools

AssetWhat to use
Brand colors Copy hex values from the Colors section below and paste into Canva's Brand Kit color fields.
Font Search Sora in Canva's font picker — it's available via Google Fonts. Use weights 600, 700, or 800 for headings; the gradient wordmark uses 700.
Logo / icon Upload images/ytdlfin-icon.png from the repo root — it's the full-resolution source (1254×1254 px) and scales cleanly to any size.
Banner image Use images/ytdlfin-banner.png from the repo root — it's the full-resolution source (1774×887 px, 1.4 MB).

These tokens appear in two places and must be kept in sync:

  • docs/ytdlfin.css — shared stylesheet for all documentation pages
  • ytdlfin/templates/base.html — inline <style> block used by the live web app

When changing a color value, update it in both files. The app also references --yt-muted for status row borders, --yt-blue for the downloading-status indicator, and --yt-purple for the focused/active form and nav states.

Submitting a download

Body — Pico CSS system stack (no custom body font)

Paste any yt-dlp-supported URL into the form on the main page. Choose a category, optionally enter a custom title, and select a quality. Hit Queue download.
