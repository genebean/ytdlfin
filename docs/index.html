ytdlfin

Self-hosted video downloader for Jellyfin — paste a URL, pick a category, done.

ytdlfin is a lightweight web app that downloads any yt-dlp-supported URL directly into your Jellyfin library. It handles folder structure, NFO metadata, and poster art automatically. Authentication is via PocketID (OIDC) with group-based access control.

User guide

Submitting a download

Paste any yt-dlp-supported URL into the form on the main page. Choose a category (set up by an admin), optionally enter a custom title, and select a quality. Hit Queue download. The download appears in the active queue and refreshes automatically every few seconds.

Quality options

1080p (default) — downloads the best available video up to 1080p and merges it into MP4. Best available — downloads the highest quality stream regardless of resolution (4K, HDR, etc.).

Custom title

Leave blank to use the video's own title. If filled in, the custom title is used for the folder name, MP4 filename, and NFO <title>. The original video title is always preserved in <originaltitle> in the NFO.

History

The History page shows all your past downloads. Admins see everyone's downloads. Filter by status (pending, downloading, done, failed, skipped).

Cancelling a download

While a download is still pending (not yet started), click the ✕ button in the queue. Once a download is in progress it cannot be cancelled from the UI.

Admin guide

Managing categories

Categories map a friendly name to a filesystem path inside the Jellyfin library. Visit Admin → Category management to add, edit, or delete categories. The path must exist and be writable by the ytdlfin service user before you can save it.

Admin access

Admin status is determined at login by PocketID group membership. Configure two groups via environment variables: ADMIN_GROUP (full admin privileges — category management and full history visibility) and USER_GROUP (regular user access — submit downloads, view own history). Users not in either group are denied access after a successful PocketID login.
